# mwmbl-vanilla
A front-end for the mwmbl search engine written in vanilla javascript

## Local Development

To run the app locally, [serve the repository root directory on your localhost](https://developer.mozilla.org/en-US/docs/Learn/Common_questions/set_up_a_local_testing_server#running_a_simple_local_http_server). The app will then be accessible at `localhost:<port>/src`.
